В рамках дипломного проекта была поставлена цель протестировать веб-приложение, API и мобильное приложение "Яндекс. Самокат".

Использованы инструменты: Postman, apiDoc, DevTools, Android Studio, Charles Proxy, CygWin, PostgreSQL.

Выполнено:

I. По веб-приложению:

1. Подготовлен чек-лист из 103 проверок к экрану "Статус заказа", выполнено тестирование, обнаружено 13 багов.

2. Выделены классы эквивалентности и граничные значения, выполнено тестирование, обнаружено 15 багов.

3. Проведено исследовательское тестирование, обнаружено 12 багов.

4. Составлен 41 баг-репортов.

II. По мобильному приложению:

1. Подготовлены 40 тест-кейсов для проверок верстки и функциональности нотификации и работы приложения при отсуствии интернет-соединения.

2. Составлены 8 баг-репортов.

III. По API:

1. Подготовлен чек-лист из 64 запросов, выполнено тестирование, обнаружено 12 багов.

2. Составлены 12 баг-репортов.
